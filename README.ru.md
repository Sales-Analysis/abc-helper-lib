## Русский

### Описание
`abc-helper-lib` — библиотека на Go для ABC-анализа товарных данных.
Классифицирует товары на группы A, B и C в зависимости от вклада в общую выручку.

### Установка
```bash
go get gitlab.com/username/abc-helper-lib@latest
```
